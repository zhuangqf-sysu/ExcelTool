# ExcelTool
Excel导入导出的工具

## 安装

下载：https://github.com/zhuangqf-sysu/ExcelTool/releases/latest

安装到maven：`mvn install xxx.jar`

在maven中引用：
```
        <dependency>
            <groupId>com.zhuangqf</groupId>
            <artifactId>excel-tool</artifactId>
            <version>0.0.1-SNAPSHOT</version>
        </dependency>
```
